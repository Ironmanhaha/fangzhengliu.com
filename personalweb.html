Research Experience

> Real-time temperature monitor program                                  06/2018 ~ 07/2018

  • Project introduction
    The Alpha Magnetic Spectrometer (AMS-02) is a state-of-the-art particle physics detector designed to operate as an external module on the International Space Station (ISS). AMS-02 is composed of different subdetectors. When on obit, the AMS-02 is exposed to the space and the thermal environment inside AMS-02 is constantly changing. So it is necessary to monitor the real-time temperature inside AMS-02 and make sure it is functioning properly. The temperature monitor program is can show the real-time temperature of one subdetector in AMS-02.

  • Personal work
       -  Complete the whole program (C language programming). Including data decoding and GUI (GTK+2.0) development.

> Space-Based Ranging Radar Echo Simulator                            09/2016 ~ 03/2018

  • Project introduction
    In this project, a simulator is designed to create target echo signal with configurable movement parameters and used for Space-Based Radar performance test. The radar can detect long-distance target around the satellite platform and obtain the range and velocity measurements with high accuracy.

    The simulator contains monitor & control computer, IF signal processing platform, RF module, and power supply module. After receiving the transmitted electromagnetic signals from radar, the simulator generates corresponding echo signals with their time-varied signal characteristics based on pre-defined target information, i.e., range, velocity, and acceleration.

  • Personal work
       -  Completed the system and signal processing algrithm design of the whole project, including the design of the basic functions, performance analysis, and interface.
       -  Completed the whole hardware design of the IF signal processing platform, including devices selection, schematic design, PCB design assistance, and PCB production. Completed the performance test of the hardware platform.
       -  Completed the FPGA program design, implementing the signal simulation algorithms and the external communication interface function.
       -  Completed the RF module technical requirements design, and completed the RF module performance test.
